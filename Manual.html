Introducción

Este manual proporciona una guía completa para utilizar el Simulador de Subestación Eléctrica. Aquí se cubren los aspectos básicos y avanzados del funcionamiento del simulador.

Instalación

  1. Descarga el archivo de instalación desde nuestra página web.
  2. Ejecuta el archivo descargado y sigue las instrucciones del asistente de instalación.
  3. Una vez instalado, abre el programa desde el acceso directo creado en tu escritorio.

Primeros Pasos

  • Al abrir el simulador, verás una pantalla de inicio con varias opciones.
  • Selecciona "Nueva Simulación" para comenzar una nueva sesión de simulación.
  • Configura los parámetros iniciales de la subestación según tus necesidades.

Funcionalidades Avanzadas

El simulador ofrece una serie de funcionalidades avanzadas para usuarios experimentados. Aquí se detallan algunas de las más importantes:

  • Simulación en Tiempo Real: Permite observar el comportamiento de la subestación en tiempo real.
  • Análisis de Fallos: Herramientas para identificar y resolver fallos en el sistema.
  • Generación de Reportes: Crea reportes detallados de las simulaciones realizadas.
